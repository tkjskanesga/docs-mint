Selamat datang di platform dokumentasi resmi komunitas **Teknik Komputer dan Jaringan (TKJ) SMKN 1 Seyegan (Skanesga)**! 

Wadah ini dirancang sebagai pusat pembelajaran, berbagi berkas konfigurasi, dan dokumentasi teknologi bagi seluruh siswa, alumni, dan guru TKJ Skanesga. Di sini, Anda dapat menemukan berbagai materi mulai dari dasar-dasar jaringan, konfigurasi perangkat, administrasi sistem, hingga komputasi awan.

## Kontribusi
Dokumentasi ini bersifat open-source untuk keluarga besar TKJ Skanesga. Jika Anda ingin menambahkan modul baru atau memperbaiki kesalahan:

<Steps>
  <Step title="Fork repositori ini di GitHub dan clone ke lokal.">
    Fork repositori ini di GitHub dan clone ke lokal.

    ```bash
    git clone https://github.com/tkjskanesga/docs-mint
    ```
  </Step>
  <Step title="Isi / ganti konten yang ingin kamu kontribusiin.">
    Isi / ganti konten yang ingin kamu kontribusiin.
  </Step>
  <Step title="Buat branch baru untuk fitur/konten Anda.">
    Buat branch baru untuk fitur/konten Anda.

    ```bash
    # 1. Buat branch baru
    git checkout -b feat/<username>/<branch>
    # 2. Add file
    git add networking/overview.mdx
    # or 'git add .' (if edit all file)
    # 3. Commit pada perubahan
    git commit -m "feat: Update new content on overview"
    # 4. Push to your fork
    git remote add feat-<username> https://github.com/<username>/docs-mint.git
    git push -u feat-<username> feat/<username>/<branch> 
    ```
  </Step>
  <Step title="Kirim Pull Request (PR) ke repositori utama.">
    Kirim Pull Request (PR) ke repositori utama.
  </Step>
</Steps>
