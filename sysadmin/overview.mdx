---

System Administration (SysAdmin) adalah disiplin ilmu yang berfokus pada pemeliharaan, konfigurasi, dan pengoperasian sistem komputer yang andal, terutama server multi-pengguna.

Di kategori ini, Anda akan mempelajari:
* Virtualisasi Server menggunakan Proxmox VE (KVM & LXC).
* Sistem Operasi Linux Server (Debian, Ubuntu, CentOS).
* Konfigurasi Web Server (Nginx, Apache).
* Pengaturan Domain Name Server (Bind9) & Mail Server (Postfix, Dovecot).
* Otomatisasi tugas menggunakan shell scripting.
