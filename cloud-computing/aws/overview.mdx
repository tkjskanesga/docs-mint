## Apa itu Amazon Web Services (AWS)?
**Amazon Web Services (AWS)** adalah platform komputasi awan yang paling komprehensif dan banyak diadopsi di dunia, menawarkan lebih dari 200 layanan unggulan dari pusat data secara global.

Di kategori ini, kita akan mempelajari layanan-layanan dasar AWS yang sering digunakan di industri:
* **Amazon EC2 (Elastic Compute Cloud):** Layanan server virtual (VPS) di cloud.
* **Amazon S3 (Simple Storage Service):** Penyimpanan objek yang aman, tahan lama, dan berskala besar.
* **Amazon RDS (Relational Database Service):** Layanan database relasional terkelola untuk MySQL, PostgreSQL, dll.
* **AWS IAM (Identity and Access Management):** Pengaturan hak akses pengguna dan resource AWS secara aman.
* **Amazon VPC (Virtual Private Cloud):** Jaringan virtual terisolasi untuk meletakkan resource AWS Anda.
