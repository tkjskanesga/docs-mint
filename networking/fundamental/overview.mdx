## Apa itu Jaringan Komputer?

Jaringan komputer (computer network) adalah sebuah sistem yang menghubungkan dua atau lebih komputer/perangkat digital menggunakan media transmisi (kabel atau nirkabel) agar dapat saling berkomunikasi, berbagi sumber daya (data, printer, internet), dan bertukar data secara efisien.

## Protokol Jaringan

Protokol adalah aturan standar yang mendefinisikan bagaimana data dikirim, diterima, dan diinterpretasikan di dalam jaringan komputer. Tanpa protokol, perangkat dari vendor berbeda tidak akan bisa saling memahami komunikasi satu sama lain.

### Protokol yang Sering Digunakan

Beberapa protokol utama yang menjadi fondasi jaringan global (Internet) antara lain:
* **IP (Internet Protocol):** Mengatur pengalamatan (IP Address) dan perutean paket data.
* **TCP (Transmission Control Protocol):** Memastikan pengiriman data berlangsung andal, berurutan, dan bebas kesalahan (connection-oriented).
* **UDP (User Datagram Protocol):** Pengiriman data cepat tanpa koneksi tetap (connectionless), cocok untuk streaming video atau game online.
* **HTTP/HTTPS (Hypertext Transfer Protocol Secure):** Protokol untuk mentransfer dokumen web secara aman.
* **DNS (Domain Name System):** Menerjemahkan nama domain (seperti `google.com`) menjadi alamat IP.
* **DHCP (Dynamic Host Configuration Protocol):** Alokasi IP Address secara otomatis ke perangkat klien.

### Fungsi Protokol

1. **Addressing (Pengalamatan):** Menentukan alamat asal dan tujuan pengiriman data.
2. **Routing:** Menentukan jalur terbaik yang dilewati paket data.
3. **Flow Control:** Mengatur kecepatan transfer agar penerima tidak kewalahan.
4. **Error Recovery:** Mendeteksi dan meminta pengiriman ulang data jika terjadi kerusakan paket.

## Sejarah Singkat Jaringan Komputer

Perkembangan jaringan komputer dimulai pada akhir tahun 1960-an:
* **ARPANET (1969):** Proyek Departemen Pertahanan AS (Advanced Research Projects Agency) yang menghubungkan komputer-komputer di universitas untuk riset militer dan akademis. Ini adalah cikal bakal Internet.
* **TCP/IP Standard (1983):** TCP/IP diadopsi sebagai standar komunikasi resmi untuk ARPANET, memungkinkan berbagai jaringan independen untuk saling terhubung.
* **World Wide Web (1990):** Tim Berners-Lee menciptakan WWW, membuat informasi di internet dapat diakses dengan mudah menggunakan peramban web (browser).
